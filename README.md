# Coin-Tracker

### Bu Coin takip programı bulunulan tarihten son 30 güne kadar (opsiyonel) olan coin fiyatlarını (yaklaşık 100 e yakın) belirtmekte ve grafiksel olarak göstermektedir. 

## Kullanımı
``` python tracker.py -t [TOKEN] ```
